⚙️ เชื่อมต่อ Google Sheet (บันทึกข้อมูลถาวร ข้ามเครื่อง/ทีม)

วิธีติดตั้ง Backend:
1) สร้าง Google Sheet เปล่า 1 ไฟล์
2) เมนู Extensions → Apps Script → วางโค้ด Code.gs ที่แนบมาให้
3) Deploy → New deployment → Web app → Execute as: Me, Who has access: Anyone
4) Copy URL (ลงท้าย /exec) มาวางในช่องด้านบนแล้วกด "บันทึก"

เมื่อเชื่อมต่อสำเร็จ ข้อมูลรายวัน (SIM/New Post/FBB/Care+) ที่กรอกทุกครั้งจะถูกบันทึกลง Google Sheet โดยตรง — เปิดจากเครื่องไหน เบราว์เซอร์ไหน หรือใครในทีมก็เห็นข้อมูลเดียวกัน ไม่มีวันหาย และแก้ไขข้อมูลตรงใน Sheet ได้ด้วย

URL นี้บันทึกไว้ใน localStorage ของเบราว์เซอร์นี้เท่านั้น — เปิดจากเครื่องอื่นต้องวาง URL เดิมซ้ำอีกครั้ง (ครั้งเดียวต่อเครื่อง)
